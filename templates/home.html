Welcome to FOODIE'S 🍕

This project predicts if a food delivery will result in a customer complaint — using real-world features like delivery time, weather, packaging, and driver rating.

🚀 Project Highlights

  • ✅ Factors: Food type, Delivery duration, Packaging, etc.
  • ✅ Helps improve delivery service quality
  • ✅ Real-time prediction capability
